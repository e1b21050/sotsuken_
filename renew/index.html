使い方
「×」ボタンを押すと、ヘルプが閉じます
●：自作プログラムでの実行 ★:Pyodideでの実行
「compile」ボタン
→ 押すと、プログラムが構文解析されます●
「reset」ボタン
→ 押すと、プログラムなどすべての表示がリセットされます
「実行結果」ボタン
→ 押すと、実行結果が表示されます
「run」ボタン
→ 押すと、プログラムが実行されます★
「ステップ実行」ボタン
→ 押すと、ステップ実行が表示されます
「step」ボタン
→ 押すと、ステップ実行が1ステップ実行されます★
「<」ボタン
→ 押すと、ステップ実行が1ステップ戻ります
「>」ボタン
→ 押すと、ステップ実行が1ステップ進みます
「ループ実行」ボタン
→ 押すと、ループ実行が表示されます
「loop」ボタン
→ 押すと、ループ実行が1ステップ実行されます★
「<」ボタン
→ 押すと、ループ実行が1ステップ戻ります
「>」ボタン
→ 押すと、ループ実行が1ステップ進みます
