使い方
「×」ボタンを押すと、ヘルプが閉じます
●：自作プログラムでの実行 ★:Pyodideでの実行
「compile」ボタン
→ 押すと、プログラムが構文解析されます●
「reset」ボタン
→ 押すと、プログラムなどすべての表示がリセットされます
「一括実行」ボタン
→ 押すと、実行結果が表示されます
「run」ボタン
→ 押すと、プログラムが実行されます★
「ステップ実行」ボタン
→ 押すと、ステップ実行が表示されます
「step」ボタン
→ 押すと、ステップ実行が1ステップ実行されます★
「<」ボタン
→ 押すと、ステップ実行が1ステップ戻ります
「>」ボタン
→ 押すと、ステップ実行が1ステップ進みます
「ループ実行」ボタン
→ 押すと、ループ実行が表示されます
※次の機能は、ステップ実行で繰り返し処理があったときのみ使用可能
「loop」ボタン
→ 押すと、ループ実行が1ステップ実行されます★
「<」ボタン
→ 押すと、ループ実行が1ステップ戻ります
「>」ボタン
→ 押すと、ループ実行が1ステップ進みます
「問題へ」ボタン
→ 押すと、問題ページに遷移します
「タイトルへ」ボタン
→ 押すと、タイトルページに遷移します
